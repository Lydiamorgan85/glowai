\# GlowAI - AI-Powered Skin Analysis Platform



\## 🌟 Overview

GlowAI is an advanced AI-powered skincare platform that provides personalized skin analysis and product recommendations. Built with modern DevOps practices and production-ready architecture.



\## 🚀 Tech Stack   



\### Backend

\- \*\*Node.js + Express + TypeScript\*\* - Production-grade REST API

\- \*\*Supabase (PostgreSQL)\*\* - Cloud database with row-level security

\- \*\*JWT Authentication\*\* - Secure token-based auth

\- \*\*Bcrypt\*\* - Password hashing

\- \*\*CORS\*\* - Secure cross-origin requests



\### Frontend

\- \*\*React + TypeScript\*\* - Type-safe component architecture

\- \*\*React Router\*\* - Client-side routing

\- \*\*Axios\*\* - HTTP client with interceptors

\- \*\*CSS3\*\* - Modern responsive design



\### DevOps

\- \*\*Git/GitHub\*\* - Version control

\- \*\*TypeScript\*\* - Type safety across stack

\- \*\*Environment variables\*\* - Secure config management



\## 📋 Features Implemented



✅ User Authentication (Signup/Login)  

✅ Secure JWT token system  

✅ Password hashing with bcrypt  

✅ Protected API routes  

✅ Beautiful responsive landing page  

✅ Production-ready error handling  

✅ Request logging middleware  

✅ CORS configuration  

✅ Type-safe frontend/backend  



\## 🔧 Local Development



\### Backend

```bash

cd backend

npm install

npm run build

npm start

\# Server runs on http://localhost:5000

```



\### Frontend

```bash

cd frontend

npm install

npm start

\# App runs on http://localhost:3000

```



\## 📊 Database Schema



\- \*\*users\*\* - User accounts with encrypted passwords

\- \*\*skin\_analyses\*\* - AI skin analysis results

\- \*\*products\*\* - Skincare product database

\- \*\*skincare\_routines\*\* - Personalized routines

\- \*\*progress\_tracking\*\* - User progress over time
